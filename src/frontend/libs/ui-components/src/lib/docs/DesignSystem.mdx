## Typography

### Font Family

We use the system font stack for optimal performance and native feel:

```css
font-family: ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont,
             "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
```

### Font Sizes

| Name | Size | Line Height | Usage |
|------|------|-------------|-------|
| `text-xs` | 0.75rem (12px) | 1rem | Captions, labels |
| `text-sm` | 0.875rem (14px) | 1.25rem | Body small, form labels |
| `text-base` | 1rem (16px) | 1.5rem | Body text |
| `text-lg` | 1.125rem (18px) | 1.75rem | Lead paragraphs |
| `text-xl` | 1.25rem (20px) | 1.75rem | Section headings |
| `text-2xl` | 1.5rem (24px) | 2rem | Page titles |
| `text-3xl` | 1.875rem (30px) | 2.25rem | Hero headings |

### Font Weights

| Weight | Value | Usage |
|--------|-------|-------|
| Normal | 400 | Body text |
| Medium | 500 | Labels, emphasis |
| Semibold | 600 | Subheadings |
| Bold | 700 | Headings |

## Spacing

We use a consistent 4px (0.25rem) spacing scale:

| Name | Size | Pixels |
|------|------|--------|
| `0` | 0 | 0px |
| `0.5` | 0.125rem | 2px |
| `1` | 0.25rem | 4px |
| `2` | 0.5rem | 8px |
| `3` | 0.75rem | 12px |
| `4` | 1rem | 16px |
| `5` | 1.25rem | 20px |
| `6` | 1.5rem | 24px |
| `8` | 2rem | 32px |
| `10` | 2.5rem | 40px |
| `12` | 3rem | 48px |
| `16` | 4rem | 64px |

## Border Radius

| Name | Size | Usage |
|------|------|-------|
| `rounded-sm` | 0.125rem (2px) | Subtle rounding |
| `rounded` | 0.25rem (4px) | Buttons, inputs |
| `rounded-md` | 0.375rem (6px) | Cards, modals |
| `rounded-lg` | 0.5rem (8px) | Large containers |
| `rounded-xl` | 0.75rem (12px) | Hero sections |
| `rounded-full` | 9999px | Pills, avatars |

## Shadows

| Name | Usage |
|------|-------|
| `shadow-sm` | Subtle elevation (inputs, cards) |
| `shadow` | Default elevation (dropdowns) |
| `shadow-md` | Medium elevation (modals, popovers) |
| `shadow-lg` | High elevation (dialogs) |
| `shadow-xl` | Highest elevation (overlays) |

```css
/* shadow-sm */
box-shadow: 0 1px 2px 0 rgba(0, 0, 0, 0.05);

/* shadow */
box-shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px 0 rgba(0, 0, 0, 0.06);

/* shadow-md */
box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06);

/* shadow-lg */
box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05);
```

## Component Sizes

Most interactive components support three sizes:

| Size | Padding | Font Size | Usage |
|------|---------|-----------|-------|
| `sm` | Compact | 0.75rem | Dense UIs, inline |
| `md` | Default | 0.875rem | Standard usage |
| `lg` | Spacious | 1rem | Touch-friendly, hero |

## Focus States

All interactive elements use a consistent focus ring:

```css
/* Focus ring */
box-shadow: 0 0 0 3px rgba(249, 115, 22, 0.2);

/* Error focus ring */
box-shadow: 0 0 0 3px rgba(239, 68, 68, 0.1);
```

## Transitions

Standard transitions for interactive elements:

```css
/* Default transition */
transition: all 0.15s ease;

/* Slower transition (modals, dropdowns) */
transition: all 0.2s ease;
```

## Z-Index Scale

| Name | Value | Usage |
|------|-------|-------|
| `z-0` | 0 | Default |
| `z-10` | 10 | Sticky elements |
| `z-20` | 20 | Dropdowns |
| `z-30` | 30 | Fixed elements |
| `z-40` | 40 | Modals |
| `z-50` | 50 | Tooltips |

## Responsive Breakpoints

| Name | Min Width | Usage |
|------|-----------|-------|
| `sm` | 640px | Mobile landscape |
| `md` | 768px | Tablets |
| `lg` | 1024px | Small laptops |
| `xl` | 1280px | Desktops |
| `2xl` | 1536px | Large displays |

```css
/* Mobile first approach */
@media (min-width: 640px) { /* sm */ }
@media (min-width: 768px) { /* md */ }
@media (min-width: 1024px) { /* lg */ }
@media (min-width: 1280px) { /* xl */ }
@media (min-width: 1536px) { /* 2xl */ }
```
