# Orange Car Rental Component Library

Welcome to the Orange Car Rental UI Component Library. This library provides a comprehensive set of reusable Angular components designed for building car rental applications.

## Overview

The component library is built with:

- **Angular 21** with Standalone Components
- **Signals** for reactive state management
- **Tailwind CSS 4** for styling with custom Orange brand theme
- **TypeScript** with strict type checking
- **Storybook 8** for documentation and visual testing

## Installation

The components are part of the `@orange-car-rental/ui-components` library within the monorepo.

```typescript
import { ModalComponent, StatusBadgeComponent } from '@orange-car-rental/ui-components';
```

## Quick Start

### Using a Component

```typescript
import { Component } from '@angular/core';
import { StatusBadgeComponent } from '@orange-car-rental/ui-components';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [StatusBadgeComponent],
  template: `
    <ocr-status-badge status="confirmed" />
  `
})
export class ExampleComponent {}
```

### Using Form Components with Reactive Forms

```typescript
import { Component } from '@angular/core';
import { FormBuilder, ReactiveFormsModule } from '@angular/forms';
import { InputComponent, CheckboxComponent } from '@orange-car-rental/ui-components';

@Component({
  selector: 'app-form-example',
  standalone: true,
  imports: [ReactiveFormsModule, InputComponent, CheckboxComponent],
  template: `
    <form [formGroup]="form">
      <ocr-input
        label="Email"
        placeholder="Enter your email"
        formControlName="email"
        [error]="emailError()"
      />
      <ocr-checkbox
        label="Subscribe to newsletter"
        formControlName="newsletter"
      />
    </form>
  `
})
export class FormExampleComponent {
  private fb = inject(FormBuilder);

  form = this.fb.group({
    email: ['', [Validators.required, Validators.email]],
    newsletter: [false]
  });

  emailError = computed(() => {
    const email = this.form.get('email');
    if (email?.hasError('required') && email?.touched) {
      return 'Email is required';
    }
    if (email?.hasError('email') && email?.touched) {
      return 'Please enter a valid email';
    }
    return null;
  });
}
```

## Design Principles

### 1. Standalone Components
All components are standalone and can be imported directly without NgModules.

### 2. Signal-Based State
Components use Angular Signals for reactive state management, providing better performance and cleaner code.

### 3. Accessible by Default
Components follow WAI-ARIA guidelines and support keyboard navigation.

### 4. Configurable & Themeable
Components accept configuration inputs for customization while maintaining the Orange brand identity.

### 5. Reactive Forms Integration
Form components implement `ControlValueAccessor` for seamless integration with Angular's reactive forms.
